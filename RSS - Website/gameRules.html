GAME RULES

A game of Reload Shield Shoot consists of a series of turns where each player chooses one of these three moves...

  • Reload
    Add one to your number of bullets. (You start with no bullets.)
  • Shoot
    To do this, you must spend a bullet. If you shoot your opponent while they are reloading, you win the game! If both players shoot each other simultaneously, the game continues.
  • Shield
    Block a shoot move from your opponent.
If a game goes on for 100 turns, then it ends in a draw.

Match play
A full match consists of multiple games, until one player reaches 10 points. The winner of each game gets a point, and if there is a draw, both players get a point. Across the entire match, both players have a maximum of 100 shields. If you run out of shields, you can only reload or shoot.
